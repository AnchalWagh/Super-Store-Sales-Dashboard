# Super-Store-Sales-Dashboard
This project is an interactive sales dashboard built using Power BI. It helps analyze sales, profit, quantity, and delivery trends across regions, segments, and categories.
## Features
- Sales, Quantity, and Profit KPIs
- Sales by Segment, Category, Sub-Category, and Payment Mode
- Monthly sales trends comparison (2019 vs 2020)
- Sales by Ship Mode and Region

## Technologies Used
- Power BI
- DAX
- Data Visualization

## Dataset
The dataset used is a Super Store Sales dataset containing sales, profit, and delivery information for multiple regions and products.

## How to Run
1. Open the Power BI Desktop application.
2. Load the dataset (`SuperStore_Sales_Dataset.xlsx` or your source file).
3. Open the provided `.pbix` file.
4. Interact with the dashboard visualizations.
